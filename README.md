# Ethereum Hackbox

Vagrant box with Ubuntu and 1337 Ethereum hacking tools.

## Base image

Uses the [gingeleski/ubuntu18_python](https://app.vagrantup.com/gingeleski/boxes/ubuntu18_python) image:

- Ubuntu 18.04
- Python 3
- Git

## Tools

- Mythril OSS
- Manticore
